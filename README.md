# Python Project_1 - Web Page Information Extractor

## Description
This Python program takes a URL from the command line,
fetches the webpage, and prints:

- Page Title (without HTML tags)
- Page Body text
- All hyperlinks present in the page

##  Technologies Used
- Python 3
- requests
- BeautifulSoup (bs4)
